**Project:** Ad Scheduler — TV & Radio (HTML/CSS/JS)  
Collaboration with Marketing Clients **P48 Marketing Company**  
**Group project by:** Sonia Singh, Connor Stimptson, Garret Fitorio and Bria Dean


  
**Features:**
- Single-page app using HTML, CSS and  JavaScript
- Create multiple named scheduler profiles
- Dayparts (e.g., Morning, Midday) with editable rates per time block
- Add ad spots with station, start/end dates, selected weekdays, daypart, ads per day, duration (seconds), and price per spot
- Select multiple weeks individually for targeted scheduling or exclusions
- Live calculations and breakdowns: daily, weekly, monthly, yearly, plus daypart and station summaries
- Calendar and list views with inline edit, delete, and instant recalculation
